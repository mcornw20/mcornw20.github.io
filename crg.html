WEBSITE REDESIGN

The goal for this homepage redesign was to create a landing page that would inform new customers about the product, present the prior wall of SEO text in a digestible manner, and to make the browsing experience more accessible. This redesign was implemented and designed by me.

Using CSS, I coded and designed a dropdown mega-menu that would replace the long scrolling sidebar of links. I took that opportunity to cut down the number of links in the top navigation, to make sure users weren’t overwhelmed by the number of options. In that dropdown menu, I also included a link to the reward points, which was a program that other retail competitors don’t offer.

Since high-quality photography of the various ornaments was limited, I came up with the top-down spread design for the sliders. The dimensions of the sliders on mobile were making the text hard to read, so I also created a square version of these sliders as well. I had to use Bootstrap to work around the hosting platform (Americommerce) not allowing the use of media queries for some reason.

The section that contains the different categories for ornaments was another feature that I was quite proud of. The previous homepage was a slider, a keyword description, and then categories all the way down. I simplified the way the text was displayed, overlapping the images and text for an engaging hover reveal that didn’t get rid of any previous SEO keywords.

Overall, the redesign gave the website a sophisticated and modern feel. I was able to successfully organize the existing information and present it in a thought-out and fun manner.

RESPONSIVE MOBILE MENU

For the mobile menu, the goal was to present all the information in the mega-menu in an easy to browse manner. Construction of this was made possible through CSS.

The multiple links found in the mega-menu were organized according to the letter of the alphabet. This was to make sure that the user would not have to scroll through multiple options while browsing on their phone, having to search for a specific link. Just like the top navigation on the desktop, the most popular links are shown as soon as the person opens the menu.

The menu button moves from its shown location to the top corner of the screen when the user scrolls. This is to make sure that it is not overlapping any imporatnt information. That same button transforms from the three lines to the back arrow, depending if the menu is open or not. Anywhere in the grayed-out area can be clicked to close out the menu, or go up a level.

Previously, the mobile menu for this website was difficult to nevigate. It was a dropdown that you tapped that took you to a huge stack of links to select from. It required too much scrolling and it also became convoluted due to a nested dropdown to show the contents of the sidebar, which had repeating content. Creating this mobile menu from scratch was an engaging task that brings consistency to the user experience.
